# 需求对照表

本文档用于对照《计算机设计大赛》项目设计文档与实际开发需求的差异。

---

## 一、前端功能需求

### PDF 设计要求

根据项目设计文档，前端应包含以下模块：

| 模块 | 功能描述 |
|-----|---------|
| 视频输入与场景管理 | 摄像头实时流、本地视频文件导入、房间/用户/监测时段配置 |
| 实时监控 | 实时画面显示、匿名化处理展示 |
| 事件日志与可视化 | 事件记录查看、日/周统计、趋势图表、风险分布 |
| 告警中心 | 管理端告警展示、紧急事件卡片 |
| 用户权限管理 | 管理员、家属/监护人、访客角色管理 |

### 实际开发需求

| 模块 | 状态 | 说明 |
|-----|------|------|
| 视频检测 | ✅ 已实现 | 本地视频文件导入并检测 |
| 实时监控 | ⏳ 开发中 | 摄像头实时流监控与检测 |
| 其他模块 | ⏸️ 待定 | 后续根据需要再补充 |
