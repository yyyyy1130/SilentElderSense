# 作品信息概要表 - 正文内容

## 一、作品简介（100字以内）

静默守护是一款面向独居老人的隐私保护型智能监护系统，采用YOLO11深度学习模型进行人体姿态检测，结合规则引擎实现跌倒、长时间静止、夜间异常三类事件的识别与四级风险评估。系统全程本地处理，人脸自动模糊，仅传输事件摘要，兼顾安全守护与隐私保护。

（共97字）

创新点：（1）隐私优先架构——本地处理、人脸模糊、仅传事件摘要；（2）规则+ML融合——YOLO11检测与规则引擎分级结合；（3）四级风险分级——匹配差异化响应策略；（4）完整系统化——从采集到响应的全链路实现，具备实际部署能力。

## 三、特别说明

### 1. 地图说明
本作品不涉及疆域地图。

### 2. 前期基础
本作品无前期基础，全部内容均为本次参赛原创开发。主要工作包括：系统架构设计、YOLO11模型部署与调优、风险引擎规则设计、前后端全栈开发、测试验证。

### 3. AI辅助工具使用
本作品使用AI辅助工具进行代码编写辅助：
- **工具名称**：Claude（Anthropic）、GitHub Copilot
- **来源**：商用AI助手服务
- **合规性**：工具用于代码编写辅助和文档润色，所有核心算法设计、系统架构、业务逻辑均为团队成员自主完成
- **生成内容占比**：AI辅助生成的代码片段约占整体代码量的20%，主要用于前端UI组件编写、API接口模板、文档格式调整等辅助性工作

### 4. 代码与数据上传
- **源代码**：已上传至比赛平台，包含前端（Vue3）、后端（Quart）、检测核心（YOLO11 ONNX）完整代码
- **模型文件**：YOLO11 ONNX模型文件已包含在源代码仓库中
- **数据集**：测试视频使用公开跌倒检测数据集，来源已在作品报告中注明
- **开源依赖**：项目依赖的开源库（Vue、Element Plus、ONNX Runtime等）通过requirements.txt和package.json声明，可自动安装
