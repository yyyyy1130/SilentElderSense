## 表格字段结构

### 第一部分：作品基本信息

| 字段名 | 说明 |
|--------|------|
| 作品编号 | 待填写 |
| 作品名称 | 待填写 |
| 作品大类 | 固定值：人工智能应用 |
| 作品小类 | 选择：□ 实践赛 / □ 挑战赛 |

### 第二部分：作品描述

| 字段名 | 字数限制 | 说明 |
|--------|----------|------|
| 作品简介 | 100字以内 | 作品简要介绍 |
| 创新描述 | 100字以内 | 作品创新点说明 |

### 第三部分：特别说明（必填项提示）

表格中"特别说明"字段包含以下提示内容：

1. 作品中如有涉及疆域的地图，请说明来源，并标注有效的地图审图号；
2. 作品如有前期基础请具体说明，并注明本次参赛的主要工作；
3. 作品如使用人工智能辅助工具进行设计与开发，请具体说明使用人工智能辅助工具的名称、来源及合规性，以及所生成的内容及占整个作品的比例。
4. 作品报告、源代码、非公开自有数据集、模型等必须上传，对于开源、企业提供的代码、数据集、模型需提供下载地址。

## 待填写字段清单

以下字段需要用户填写：

1. **作品编号** - 系统生成或比赛组织方分配
2. **作品名称** - 如：SilentElderSense（智慧养老监护系统）
3. **作品小类** - 选择实践赛或挑战赛
4. **作品简介** - 100字以内
5. **创新描述** - 100字以内
6. **特别说明** - 根据实际情况填写
7. **作者分工表** - 填写每位作者姓名及分工百分比
8. **指导教师作用** - 描述指导教师贡献
